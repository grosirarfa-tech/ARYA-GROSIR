Kode Script Backend (Google Apps Script) & Panduan Setup

  1. Buka Google Sheets baru, beri nama Database POS ARYA GROSIR.
  2. Buka menu Ekstensi > Apps Script.
  3. Hapus semua kode bawaan, lalu salin (copy) kode Apps Script di bawah ini secara utuh.
  4. Klik Terapkan (Deploy) > Penerapkan Baru (New Deployment).
  5. Pilih Jenis: Aplikasi Web (Web App). Akses (Who has access): Siapa Saja (Anyone).
  6. Salin Web App URL yang dihasilkan, lalu tempelkan di kolom input URL di atas.
