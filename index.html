Panduan Pengaturan Google Sheets

  1. Buat Spreadsheet baru di Google Sheets.
  2. Pastikan untuk membuat sheet/tab dengan nama berikut: STOCK_LIST, PROJECTION, PROFIT, EXPENSES, dan SALES.
  3. Klik menu Ekstensi > Apps Script.
  4. Hapus semua kode bawaan lalu tempel kode script di bawah ini.
  5. Klik tombol Simpan (ikon disket), lalu terapkan sebagai Aplikasi Web untuk semua orang.
