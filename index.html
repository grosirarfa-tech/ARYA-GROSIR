Google Sheets Sync

Konfigurasi database cloud cadangan terpusat

Cara Setup Script Database Pusat

  1. Buat Google Spreadsheet baru di Drive Anda.
  2. Klik menu Ekstensi → Apps Script.
  3. Hapus kode bawaan, lalu salin-tempel kode Apps Script terpadu di bawah ini:
function doPost(e) {
  var token = "ARYA_GROSIR_SECURE_SYNC";
  var contents = JSON.parse(e.postData.contents);
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  
  // 1. STOK BARANG (Dukungan Kolom Satuan & Target Stok)
  var sheetStok = ss.getSheetByName("STOK_BARANG") || ss.insertSheet("STOK_BARANG");
  sheetStok.clear();
  sheetStok.appendRow(["Barcode", "Nama Barang", "Sisa Stok", "Harga Beli", "Harga Jual", "Satuan", "Target Stok"]);
  contents.stok.forEach(function(r) {
    sheetStok.appendRow([r.barcode, r.nama, r.stok, r.hargaBeli, r.hargaJual, r.satuan || "pcs", r.targetStok || 0]);
  });
  
  // 2. TRANSAKSI PENJUALAN
  // ... (lanjutan script sync Anda)
}

4. Klik Terapkan (Deploy) → Penerapan Baru. Pilih jenis "Aplikasi Web", setel akses ke "Siapa saja", lalu salin URL yang dihasilkan ke form di atas.
